# Simple Covenants and Tools for GNARK

```text
A cli for generating doge covenant proofs using the BLS12-381 curve.

Usage:
  doge-covenant [command]

Available Commands:
  bitide              Generate a BitIDE verifier script for a proof/vk
  completion          Generate the autocompletion script for the specified shell
  convert             Generate a BitIDE verifier script for a proof/vk
  help                Help about any command
  provedummy          Prove the example circuit
  provesha256         Prove the simple sha256 circuit
  provesimplecovenant Prove the simple covenant circuit
  verify              Verify a proof
  ```
